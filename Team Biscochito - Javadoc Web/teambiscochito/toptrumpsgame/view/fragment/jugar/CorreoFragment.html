        public void validarCorreo(java.lang.String porcentajeAciertos,
                          java.lang.String correo,
                          java.lang.String subject)

        Team Biscochito

        Método que se ejecuta para validar el correo por si el EditText del correo está vacío o es un correo inválido. También tiene en cuanta que si no hemos respondida ninguna pregunta la plantilla del correo a enviar cambia. Una vez hecho esto se mostrará un diálogo para verificar: correoDialogConfirmar()

        Referencia del método en: viewCorreoEnviar()

        Parameters:
        porcentajeAciertos - el porcentaje de aciertos.
        correo - el correo del EditText.
        subject - String de el asunto del correo.
